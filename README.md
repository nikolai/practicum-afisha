# Анализ лояльности пользователей Яндекс Афиши

Исследовательский анализ данных для выявления характеристик лояльных пользователей платформы.

## Запуск проекта

### Для Fish shell:

```fish
fish start.fish
```

### Для Bash/Zsh:

```bash
./start.sh
```

После запуска откройте `module_2_project.ipynb` и выберите kernel `Python (Sprint 8 Afisha)`.

## Конфигурация

Создайте файл `.env` в корне проекта с параметрами подключения к базе данных PostgreSQL:

```env
DB_USER=...
DB_PASSWORD=...
DB_HOST=...
DB_PORT=...
DB_NAME=...
```

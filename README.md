# Анализ лояльности пользователей Яндекс Афиши

Исследовательский анализ данных для выявления характеристик лояльных пользователей платформы.

## Требования

- **Python**: 3.9 или выше
- **PostgreSQL**: 12 или выше
- **Свободное место**: ~500MB (для зависимостей и данных)
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

### Шаг 1: Установка зависимостей

Создайте файл `.env` на основе шаблона `.env.example`:

```bash
cp .env.example .env
```

Затем отредактируйте `.env` и укажите ваши данные для подключения к PostgreSQL.

### Шаг 2: Запуск скрипта установки

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```
